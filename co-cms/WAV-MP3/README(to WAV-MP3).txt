Сборка:
Впапке с леймом лежит решение lame_vc8.sln, запускаем его.
Выгружаем из решения проект mp3x_vc8, и загружаем проект Example
он лежит в \lame-398-2\Dll
далее правый клик по решению Project Dependecies и там выставляем, что 
Example зависит от LameDLL_vc8.
далее собираем LameDLL_vc8,потом собираем только проект Example, 
потом уже решение.

Запуск:
Проделав успешно вышеуказанное, в директории  lame-398-2\Dll\Release
появляется Example_vc6.exe
Его следует запускать из командной строки с параметром - именем входного файла.
(Входной файл имеет расширение .wav и лежит в папке с экзешником)
или просто вызовите - как пользоваться она сама напишет

Комментарии:
В проекте Example есть разделение по файлам (сорс и хэдер) нашего класса
программка мейн - просто пример.
Главная функция класса ( .Encode() ) пока не имеет аргументов, кроме
имени входного файла. Все важные параметры кодирования указаны в теле функции

Выкладываю только проект Example с исходниками 
(исходники класть также в \lame-398-2\Dll ).
